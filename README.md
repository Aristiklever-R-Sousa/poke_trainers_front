# Wellcome, Trainer!!

Esse projeto tem como objetivo formar um time pokemon com até no máximo 5 pokemons!

Como recursos, temos:
- Login
- Cadastro
- Listagem de Pokemons
- Pesquisa por pokemons (nome ou tipo)
- Adição de Pokemons ao time
- Visualização dos seus Pokemons

### Commands

```$ yarn dev```
